# Gatus (pbb-status)

Service monitoring. Third-party open-source tool for uptime monitoring.

## Deployment

Not deployed by the monorepo. Runs on a completely separate server for more precise monitoring (monitors from outside the application infrastructure).  
By the way, its configuration is stored in the monorepo repository.  

## Purpose

- Monitor availability of API, WebApp, Admin Panel, CDS and other services
- Alert on downtime
- Status page (pbb-status) for public or internal visibility
