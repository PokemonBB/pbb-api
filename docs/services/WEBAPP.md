# WebApp

Main web application (game). Frontend for the PBB ecosystem.

## Stack

- Svelte, TypeScript
- Phaser 3 (game graphics and physics)
- Static Adapter (SPA) + Nginx in Docker Hub image

## Purpose

- User-facing game interface
- Login, registration, account activation
- User profile and configuration
- Friends management
- Consumes the API and CDS for data and content
