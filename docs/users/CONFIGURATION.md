# Personal Configuration

Each user can customize their experience through the configuration object.

## Language

`es` (Spanish) or `en` (English)

## Theme

`system` (follow OS), `dark` or `light`

Users update their configuration via:

- Main game: [Main Game]({WEBAPP_BASE_URL})
- API endpoint: [Update current user configuration]({API_BASE_URL}/api#/Users/UsersController_updateConfiguration)
